Gazetteer - Details
  • Overview:
    • Interactive map application using Leaflet.js (open-source JavaScript library for maps).
    • Backend powered by PHP with AJAX for smooth data fetching.
  • Technologies Used:
    • HTML/CSS: Structure & styling of the homepage, including the map container and UI elements.
    • Bootstrap: Ensures responsive design and layout styling.
    • JavaScript/jQuery:
      • Handles DOM manipulation & interactive features.
      • Loads Leaflet.js and additional plugins.
    • PHP: Provides an API endpoint returning geospatial data in JSON format.
    • AJAX: Retrieves data asynchronously for smooth map updates without page reloads.
    • JSON: Used for data exchange between client and server.
    • XAMPP: Used to create a local development environment with Apache, MySQL, and PHP for testing and debugging before deployment.
